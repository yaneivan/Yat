# HTR Polygon Annotation Tool (Yat) - Руководство пользователя

HTR Polygon Annotation Tool (Yat) — это веб-приложение для подготовки и аннотирования данных для распознавания рукописного текста (HTR/OCR). Приложение позволяет выполнять полный цикл подготовки данных: от кадрирования оригинальных сканов до точной полигональной сегментации текстовых строк и ввода текста. Основные возможности включают: создание и управление проектами, импорт/экспорт данных в формате ZIP с XML-аннотациями в формате PAGE, кадрирование изображений с возможностью поворота, инструменты для полигональной сегментации текстовых строк, редактор текста с поддержкой автоматического распознавания через модель TROCR, а также пакетную обработку для автоматической детекции и распознавания текста. Интерфейс интуитивно понятен и включает удобные инструменты для точной аннотации с поддержкой истории изменений.

## Импорт и экспорт данных

### Формат данных

Приложение использует формат PAGE XML для хранения аннотаций. Этот формат совместим с eScriptorium и другими инструментами для аннотации документов. При экспорте проекта создается ZIP-архив, содержащий:

- Изображения в корне архива
- XML-файлы аннотаций в формате PAGE XML в корне архива
- Файл проекта project.json с метаданными

### Особенности импорта

Приложение поддерживает импорт данных из двух источников:

1. **Импорт на главной странице**:
   - Создает новый проект с уникальным именем
   - Извлекает изображения и XML-файлы из архива
   - Обрабатывает XML-файлы и сохраняет аннотации
   - Обновляет информацию о проекте

2. **Импорт на странице проекта**:
   - Добавляет изображения и аннотации к существующему проекту
   - Сохраняет изображения в общую папку
   - Обновляет информацию о проекте

При импорте система автоматически определяет формат аннотаций и преобразует их в внутренний формат приложения. Поддерживаются как полигональные, так и прямоугольные аннотации текстовых строк.
