# 📋 План ручного тестирования Yat

## 🚀 Подготовка

```bash
# 1. Запустить приложение
uv run python app.py

# 2. Открыть браузер
http://127.0.0.1:5000
```

## 1️⃣ Управление проектами

### 1.1 Создание проекта
**Где:** Главная страница (`/`)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать "Создать проект" | Открылось модальное окно |
| 2 | Ввести название `TestProject`, описание | — |
| 3 | Нажать "Создать" | Проект появился в списке |

### 1.2 Редактирование проекта
**Где:** Страница проекта (`/project/TestProject`)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать "Редактировать проект" | Открылась форма редактирования |
| 2 | Изменить название на `NewName` | — |
| 3 | Сохранить | Название обновилось, URL изменился на `/project/NewName` |

### 1.3 Удаление проекта ⚠️
**Где:** Страница проекта

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать "Удалить проект" | Подтверждение |
| 2 | Подтвердить | Проект удалён, возврат на главную |

### 1.4 Проверка на дубликаты ⚠️
**Где:** Главная страница

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Создать проект с именем `Duplicate` | Успешно |
| 2 | Создать ещё один проект `Duplicate` | **Ошибка:** "Project already exists" |

## 3️⃣ Кадрирование (Cropper)

### 3.1 Переход в кроппер
**Где:** Страница проекта → кнопка "Crop" у изображения

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать "Crop" | Открылся `/cropper?image=filename.jpg` |
| 2 | **Проверить кнопку "Назад в проект"** | Кнопка видна ⚠️ (известный баг) |

### 3.2 Кадрирование
**Где:** Страница кроппера

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Выделить область (drag mouse) | Рамка видна |
| 2 | Повернуть рамку (rotation handle) | Рамка поворачивается |
| 3 | Нажать "Crop" | Изображение обрезано, возврат в проект |

### 3.3 Навигация стрелками ⚠️
**Где:** Страница кроппера

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать `→` (следующее изображение) | Загрузилось следующее |
| 2 | **Проверить кнопку "Назад"** | ⚠️ Кнопка пропала (известный баг) |
| 3 | Нажать `←` (предыдущее) | Загрузилось предыдущее |

### 3.4 Недеструктивность
**Где:** Страница кроппера

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|

| 2 | Снова открыть кроппер | Можно переобрезать из оригинала |

## 4️⃣ Сегментация (Editor)

### 4.1 Переход в редактор
**Где:** Страница проекта → кнопка "Segment"

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать "Segment" | Открылся `/editor?image=filename.jpg` |
| 2 | **Проверить загрузку изображения** | ⚠️ Может не прогрузиться с первого раза (баг) |

### 4.2 Рисование полигонов
**Где:** Страница редактора, режим `N` (Draw)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать `N` | Режим рисования |
| 2 | Кликнуть 4-5 раз (создать полигон) | Полигон нарисован |
| 3 | Нажать `Esc` | Рисование отменено |

### 4.3 Редактирование полигонов
**Где:** Страница редактора, режим `V` (Select)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать `V` | Режим выделения |
| 2 | Кликнуть на полигон | Полигон выделен |
| 3 | Перетащить вершину | Вершина переместилась |
| 4 | Нажать `Del` | Полигон удалён |

### 4.4 Undo/Redo ⚠️
**Где:** Страница редактора

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Создать полигон | — |
| 2 | Нажать `Ctrl+Z` | Полигон удалён |
| 3 | Нажать `Ctrl+Shift+Z` | Полигон восстановлен |
| 4 | **Проверить баг** | ⚠️ Иногда отменяет несколько действий сразу |

### 4.5 Сохранение
**Где:** Страница редактора

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нарисовать 1-2 полигона | — |
| 2 | Нажать "Save" | Статус: "Сохранено" |
| 3 | Обновить страницу (`F5`) | Полигоны на месте |

### 4.6 Навигация
**Где:** Страница редактора

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать `Ctrl+→` | Следующее изображение |
| 2 | Нажать `Ctrl+←` | Предыдущее изображение |
