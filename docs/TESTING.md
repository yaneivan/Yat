# 📋 План ручного тестирования Yat

## 🚀 Подготовка

1. Запустить приложение: `uv run python app.py`
2. Открыть браузер: `http://127.0.0.1:5000`

---

## 📖 Сценарий 1: Полный цикл работы с проектом

### Шаг 1: Создание проекта

**Где:** Главная страница (`/`)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать **«+ Создать проект»** | Открылось модальное окно |
| 2 | Ввести название `TestProject`, описание | — |
| 3 | Нажать **«Создать»** | Проект появился в списке |
| 4 | Проверить счетчик на карточке | Показывает `0 изображений` |

---

### Шаг 2: Загрузка изображений

**Где:** Страница проекта → область загрузки

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать на область **«Перетащите изображения сюда или нажмите для выбора»** | Открылся диалог выбора файлов |
| 2 | Выбрать 2-3 изображения (jpg, png) | — |
| 3 | Файлы загрузятся автоматически | Изображения появились в списке |
| 4 | Проверить счетчик на карточке | Показывает `N изображений` |

---

### Шаг 3: Кадрирование (Cropper)

**Где:** Страница проекта → кнопка **«Обрезка»** у изображения

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать **«Обрезка»** у изображения | Открылся `/cropper?image=filename.jpg` |
| 2 | Выделить область (drag mouse) | Желтая рамка видна |
| 3 | Повернуть рамку (rotation handle) | Рамка поворачивается |
| 4 | Нажать **«✂️ Сохранить»** | Изображение обрезано, возврат в проект |
| 5 | Проверить навигацию стрелками | `←` / `→` переключают изображения |

**Недеструктивность:** можно снова открыть кроппер и переобрезать из оригинала.

---

### Шаг 4: Сегментация (Editor)

**Где:** Страница проекта → кнопка **«Сегменты»** у изображения

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать **«Сегменты»** | Открылся `/editor?image=filename.jpg` |
| 2 | Нажать `N` (режим рисования) | Курсор изменился |
| 3 | Кликнуть 4-5 раз (создать полигон) | Полигон нарисован |
| 4 | Нажать `V` (режим выделения) | Можно двигать полигон |
| 5 | Перетащить вершину | Вершина переместилась |
| 6 | Дождаться «Сохранено» | Автосохранение сработало |
| 7 | Обновить страницу (`F5`) | Полигоны на месте |

**Навигация:**
- Стрелки `←` / `→` в верхней панели — переключают изображения

**Undo/Redo:**
- `Ctrl+Z` — отменить действие
- `Ctrl+Shift+Z` — повторить действие

---

### Шаг 5: Распознавание текста (Text Editor)

**Где:** Страница проекта → кнопка **«Распознавать»** у изображения

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Нажать **«Распознавать»** | Открылся `/text_editor?image=filename.jpg` |
| 2 | Кликнуть на полигон слева | Открылось модальное окно с текстом |
| 3 | Отредактировать текст | Текст обновился |
| 4 | Нажать **«Сохранить и следующий (Enter)»** | Текст сохранён, переход к следующему региону |
| 5 | Обновить страницу | Текст на месте |

## 📖 Сценарий 3: Мониторинг задач

**Где:** Браузер → `http://127.0.0.1:5000/api/tasks`

| Поле | Ожидаемое значение |
|------|-------------------|
| `id` | UUID |
| `type` | `batch_detection` или `batch_recognition` |
| `status` | `pending` / `running` / `completed` / `failed` |
| `progress` | 0-100 |
